What is advanced data analysis?

  • Envisioned in part as teaching students how NOT TO DO crappy science (or how TO DO good science)
  • In our program, first year classes (biostats, epi methods, ALM) provide an introduction to different models used in statistical analysis without getting much into all the different components that go into producing a solid statistical evaluation of a public health problem.
  • AND of course all along the way much experience is obtained with statisitcal software (R in this case) and data management

What consitutes good science? (My top 5)

  1. A reproducible work flow so that 10 years from now or even just 6 months from now you or someone else can reproduce the results from a publication or report, exactly. What information do you need to provide to help ensure reproducibility?

  2. Knowledge of the main statistical models used to analyze public health problems, when they are appropriate, when they are not appropriate, what you should do to assess model fit/diagnostics.

  3. Adjusting for covariates using DAGs. We SHOULD NOT throw the kitchen sink in the model. This is a tricky business and we need to use a systematic way of covariate selection based on logic and practical issues.

  4. Robustness checkswhen uncertain of what the best model is, do it multiple ways, also do sensitivity analyses to see if conclusions change based on your CHOICE of how to model the data.

  5. Accurate communication of findings

CLASS FIVE: Logistic regression

  • Dependent variable: Binary variable
  • Independent variable: Categorical and continuous
  • Measure of Association: Odds ratio (OR) (exp of B from the model)
  • Model assumptions: Several
  • What does the model predict?
    • The probability that an observation falls into one category given the independent variables
    • The log odds that an observation falls into one category given independent variables
  • Model things to check:
    • liklihood ratio test
      • deviance = -2 X log-likelihood
      • null deviance = model without any predictors
      • residual deviance=model with predictors
      • liklihood ratio test = null deviance - residual deviance = chi-square with df from base model - df from full model (i.e.with additional predictor(s))

*Note to compare models you need to make sure they include the same number of observations (nested models)

  • Sensitivity and specificity of the model (model fithow well does the model do at classifying people correctly)
  • How do we interpret the results? Go to class 5 demo

CLASS SIX: Multinomial logistic regression

  • Dependent variable: Categorical variable with J levels
  • Independent variable: Continuous or categorical
  • Measure of Association: Odds ratio (OR) (exp of B from the model)
  • Model assumptions: several
  • What does the model predict?
    • The probability that an observation falls into category J given independent variables
    • The log odds of that an observation falls into category J given independent variables
  • Model fit-Can compare two models using the Liklihood ratio test
  • How do we interpret the results?
    • The effect of independent variables on the probability of category J, in comparison to the reference category.

CLASS SIX: Ordinal logistic regression

  • Dependent variable: ordered categorical variable with J levels
  • Independent variable: Continuous or categorical
  • Measure of Assocation: Odds Ratio (exp of B from the model)
  • Model assumptions: proportional odds assumption (can be tested by comparing deviances between multinomial and ordinal model, from pg. 272 Fox and Weisberg An R companion to Applied Regression 2nd edition)

CLASS SEVEN: Poisson regression

  • u is known as the rate and is the expected number of times that an event has occurred per unit of time.
  • Model assumptions:
    • Conditional variance equals the conditional mean
  • Dependent variable: Count
  • Independent variables: Continuous or categorical
  • Measure of association: Incidence rate ratio (IRR) (exp of B from the model)
  • How do we interpret the results?
    • For a unit change in xk, the expected count changes by a factor of expBk, other things being equal
  • What to watch out for: Overdispersion, lots of 0s (zero inflated), no 0s (zero truncated)

CLASS SEVEN: Negative Binomial regression

  • Use when there is overdispersion
  • Model assumptions:
    • Conditional variance equals the conditional mean plus an overdispersion parameter (alpha) multiplied by a quadratic term of the mean
  • Dependent variable: Count
  • Independent variables: Continuous or categorical
  • Measure of association: Incidence rate ratio (IRR) (exp of B from the model)
  • How do we interpret the results? Same as Poisson
  • What to watch out for: lots of 0s (zero inflated), no 0s (zero truncated)

Marginal effects (all models)

  • See https://cran.r-project.org/web/packages/margins/vignettes/TechnicalDetails.pdf for more reading
  • Marginal effects at representative values (MERs): Fitted values at representative values of X
  • Marginal effects at the mean of X (MEMs): not particularly useful
  • Average marginal effects (AMEs): calculates a fitted value y-hat for every observed value of X and then averages them
